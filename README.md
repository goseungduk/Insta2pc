# insta2pc
K-Instagram pics downloader

인스타그램 사진 다운용 v1.04.18.

---

## **기능**

- 하나의 게시글에 대한 모든 사진 다운로드
- 하나의 계정에 대한 모든 썸네일 다운로드
- 하나의 계정에 대한 모든 사진 다운로드
